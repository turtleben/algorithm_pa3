------------------
-- Introduction --
------------------

AlgParser API provides a few functions to parse the 
inputs/outputs of Program assignment #3.


-----------------
-- Quick Start --
-----------------

To compile the sample program of using AlgParser, use command:

	g++ -O2 main.cpp libparser.a -o parser

To run the binary, use command:

	./parser [input_file_name]

	e.g. ./parser ../benchmarks/gr4x4.in
